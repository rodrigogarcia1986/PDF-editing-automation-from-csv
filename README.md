# Script de Automação de E-mail | Email Automation Script

Este é um script em Python projetado para automatizar o processo de envio de arquivos PDF personalizados para uma lista de destinatários por e-mail. O script lê dados de um arquivo CSV, processa esses dados, gera PDFs personalizados, envia esses PDFs como anexos de e-mail e registra o status dos e-mails enviados em um arquivo JSON.

This is a Python script designed to automate the process of sending personalized PDF files to a list of recipients via email. The script reads data from a CSV file, processes it, generates PDFs with customized content, sends these PDFs as email attachments, and logs the status of the sent emails in a JSON file.
